Wiki Markup Help

Paragraphs

  • A new text paragraph is created whenever two blocks of text are separated by one or more empty lines.
  • Writing "~" at the end of a line inserts a line break.

Font Styles

-'''bold'''
-''italic''
-__strike-through__
Display:
  • bold
  • italic
  • strike-through

Preformatted Text

In order to mark a paragraph as preformatted text, indent that paragraph with one or more spaces.

 public static void main(String[] args) {
   System.out.println("Hello world!");
 }
Display: 
 public static void main(String[] args) {
   System.out.println("Hello world!");
 }

Preformatted Text With Syntax Highlight

In order to mark a paragraph as preformatted source code text, add tags #pre and #code to your fragment.

 public static void main(String[] args) {
   System.out.println("Hello world!");
 }
Display: 
 public static void main(String[] args) {
   System.out.println("Hello world!");
 }

Blockquotes

In order to mark a paragraph as blockquote, start the line of that paragraph with '>'

>This text is a quote from someone else.
>>You can create a nested quote.
Display:
This text is a quote from someone else.
You can create a nested quote.

Links

  • Hyperlinks are automatically created for tag names and URLs
  • An image tag is embedded if a URL ends with image extension(gif|png|jpeg|jpg).
  • An video player is embedded if a URL is a Youtube video.
Fragment Link: 
-#1
-fragment:1
-fragment:1:title
-fragment:1:detail
Display:
  • #1
  • fragment:1
  • Fragment Title
  • #1 Fragment Title
Labeled Link: 
-[http://www.google.com/ Google]
-[fragment:1 Link to the fragment #1]
Display:
  • Google
  • Link to the fragment #1

Embed the content of another fragment

You can embed the content of another fragment. 

fragment:1:embed
